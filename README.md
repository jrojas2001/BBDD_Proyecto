# BBDD_Proyecto
Muestra Base de datos proyecto integrado


## Scripts incluidos
- `sql/schema.sql`: Estructura de la base de datos
- `sql/seed.sql`: Datos de ejemplo

## Cómo restaurar la base de datos
1. Instala PostgreSQL en tu PC
2. Instala PGAdmin para administrar visualmente tu base de datos Postgre
3. Crea la base de datos: Ej. Gastronomía
4. Copia el codigo de Schema.sql, pegalo en tu pgadmin
5. Ejecuta el script y tendrás tu base creada

## Notas
No se tiene acceso directo a la base de datos real. Estos scripts son suficientes para recrearla localmente.
